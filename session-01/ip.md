이번에는 **IP 프로토콜**에 대해서 알아보겠습니다.

예를 들어

제가 미국에 있는 친구에게

```
Hello World
```

라는 메시지를 보내려고 합니다.

이때 최소한 다음 정보가 필요합니다.

- 나의 주소
- 친구의 주소

인터넷에서도 마찬가지입니다.

그래서 모든 컴퓨터에는

**IP 주소**라는 것이 있습니다.

예를 들어

클라이언트의 IP 주소가

```
100.100.100.1
```

서버의 IP 주소가

```
200.200.200.2
```

라고 가정해 보겠습니다.

---

IP 프로토콜은

> **이 IP 주소를 이용해
데이터를 목적지까지 전달하는 규칙**
> 

입니다.

데이터는 **패킷(Packet)** 이라는 단위로 전달됩니다.

패킷에는 다음과 같은 정보가 들어갑니다.

- 출발지 IP 주소
- 목적지 IP 주소
- 전송 데이터

즉,

```
출발지 IP
목적지 IP
데이터
```

이 정보를 담아서 인터넷으로 보내면

네트워크 장비들이 목적지를 확인하고

패킷을 다음 노드로 전달합니다.

이 과정이 반복되면서

결국 패킷은 **목적지 서버까지 도달하게 됩니다.**
