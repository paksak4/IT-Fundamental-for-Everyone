## Port

다음은 **Port**입니다.

포트는 쉽게 말하면

> **하나의 컴퓨터 안에서
어떤 프로그램과 통신할지 구분하는 번호**
> 

입니다.

예를 들어

하나의 PC에서

- 게임
- 화상 통화
- 웹 브라우저

를 동시에 사용한다고 생각해 보겠습니다.

이때 서버에서 응답이 오면

> 어떤 프로그램이 받아야 하는지
> 
> 
> 구분해야 합니다.
> 

이때 사용하는 것이 바로 **Port 번호**입니다.

---

### 비유

IP = 아파트 주소

Port = 아파트 호수

즉,

```
IP → 건물
Port → 방 번호
```

라고 이해하면 됩니다.

---

대표적인 포트 예시

- HTTP → 80
- HTTPS → 443

포트 번호는

```
0 ~ 65535
```

범위에서 사용할 수 있습니다.
