# IT-Fundamental-for-Everyone

###  전체 학습 목표 
이 과정은 웹 기술을 단순히 “사용하는 법”이 아니라,  
**왜 이런 기술들이 등장했고 어떤 문제를 해결하기 위해 만들어졌는지**를 이해하는 것을 목표로 합니다.

HTTP를 출발점으로 웹의 기본 원리를 이해하고,  
이후 서버 운영과 배포, 그리고 운영 관점까지 확장하며  
**백엔드 개발자로서의 기본 사고방식과 시야를 형성**합니다.

  예시) https://github.com/daeun-ops/java-study/issues/1

- 담당자가 Github 공유 하는 방법 (정해진 담당자만)
- 링크의 repository에서 모든 학습 내용을 markdown 형식으로 기록한다.
- 섹션 별로 구분되어있는 디렉터리에 각자 read.md 파일로 push 한다.
  예시) https://github.com/daeun-ops/java-study/tree/main
- 벌칙은 따로 없으나 담당 이슈에 대한 코멘트, 과제 수행은 무조건 빠짐없이 참여한다는 것을 전제로 합니다.
